# Whispaper Releases

This repository hosts public distribution assets for Whispaper.

GitHub Pages serves the Sparkle appcast feed from:

`https://xiaohuliming.github.io/Whispaper-Releases/appcast.xml`

Typical contents:

- notarized `.zip` or `.dmg` archives attached to GitHub Releases
- `docs/appcast.xml` for Sparkle
- `docs/release-notes/` for HTML release notes
- model archives published separately as release assets

Source code lives in the private repository:

`https://github.com/xiaohuliming/Whispaper`
