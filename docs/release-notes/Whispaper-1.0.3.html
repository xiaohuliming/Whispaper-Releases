Whispaper 1.0.3

内测更新说明

本次更新

  • 修复另一台 Mac 打开后 backend 启动失败的问题。
  • 打包时会把 Python runtime 依赖的 Homebrew 动态库内嵌进 App。
  • 重写 `_sqlite3`、`_ssl`、`_hashlib`、`_lzma`、`_decimal` 等扩展的动态库路径，避免依赖 `/opt/homebrew`。
  • 已验证 App 包内不再残留 `/opt/homebrew` 或 `/usr/local` 动态库引用。

说明

本次仍然是 GitHub Releases 分发的未公证内测包。它修复的是运行时依赖缺失问题；Gatekeeper 信任提示仍需要后续 Developer ID 签名和 notarization 解决。
