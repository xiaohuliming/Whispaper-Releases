Whispaper 1.0.1

内测更新说明

本次更新

  • 修复 backend 启动卡在欢迎页的问题，避免配置字段错误导致健康检查失败。
  • 补上标准 macOS 菜单栏，恢复复制、粘贴、全选等系统编辑操作。
  • 修复“查看模型清单”错误跳浏览器下载的问题，改为应用内查看清单内容。
  • 离线转写支持本地多模型选择，新增 `Whisper Large V3 Turbo Q4`、`Medium`、`Small`、`Tiny` 等选项。
  • 新增“自定义本地模型目录”，可以把本机已有的 Whisper 兼容模型目录挂进 Whispaper 使用。
  • backend 启动改为动态端口，避免重启后复用旧进程导致前端资源和行为不一致。

说明

本次仍然是 GitHub Releases 分发的内测包，适合验证 Sparkle 更新链路和模型切换逻辑，不包含正式签名和 notarization。
