Whispaper 1.0.2

内测更新说明

本次更新

  • 新增笔记本首页和笔记本内页面，进入笔记本后可先管理参考资料与历史笔记，再开始转录。
  • 参考资料支持点击选择和拖拽上传，支持 PDF、PPTX、DOCX、Markdown 和文本文件。
  • 参考资料按内容哈希去重，重复上传会跳过并清理历史重复副本。
  • 本地 RAG 解析层升级：PDF 按页、PPTX 按幻灯片和文本块、DOCX 按标题和段落建立可引用片段。
  • 本地 RAG 索引升级为 SQLite FTS5 + 本地语义索引的双通道检索，笔记生成时更容易命中课件里的术语和事实。
  • 参考资料区默认折叠，保留说明文案，并显示叠放文件卡片预览，减少历史笔记被资料列表挤下去的问题。
  • 修复上传文献无反应、转录声纹不显示、`python-multipart` 打包缺失等问题。

说明

本次仍然是 GitHub Releases 分发的内测包，适合验证笔记本、参考资料和本地 RAG 工作流，不包含正式签名和 notarization。
