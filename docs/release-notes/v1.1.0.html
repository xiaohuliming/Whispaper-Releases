Whispaper 1.1.0

内测更新说明

本次更新

  • 新增“系统语音识别”转录引擎，可直接调用 macOS 自带语音识别，降低本地 Whisper 的持续负载。
  • 新增“在线 STT 服务商”模式，支持接入 OpenAI-compatible /v1/audio/transcriptions 接口。
  • 设置页、返回导航和转录工作区的布局做了一轮整理，界面交互更稳定。
  • 修复发布包内 backend Python 运行时依赖，避免 httpx 缺失导致服务启动失败。

说明

本次仍然是 GitHub Releases 分发的 Sparkle 内测包，不包含 Developer ID 签名和 notarization。离线 Whisper 模型仍保持在 App 外部独立安装，不会随应用更新重复下载。
